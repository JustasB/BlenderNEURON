*****
Steps
*****

Ensure that 'Requirements <https://docs.blenderneuron.org/requirements.html/>'_ are met.

1. Download the BlenderNEURON ZIP file from `Download Page <https://blenderneuron.org/download/>`_
    *
2. Extract the BlenderNEURON ZIP file into your model’s directory
    *
3. Start BlenderNEURON for NEURON
    * Load your model in NEURON
    * In NEURON Python run 'from blenderneuron import neuronstart'
    * BlenderNEURON should be running within NEURON
4. Start BlenderNEURON for Blender:
    * Install Blender 2.79: Linux | Mac | Windows
    * Open Blender 2.79. Then File > User Preferences > Add-ons > Install Add-on From File
    * Point to the ZIP file
    * Check the box next to ‘Import-Export: BlenderNEURON’
    * Click Save User Settings
    * BlenderNEURON tab should be visible on the left side of Blender
